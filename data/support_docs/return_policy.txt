# Return & Exchange Policy

## Return Eligibility

### Timeframe
- Standard returns: Within 30 days of delivery
- Extended holiday returns: Items purchased Nov 1 - Dec 31 can be returned until Jan 31
- Defective items: Within 90 days of delivery

### Condition Requirements
Items must be returned in:
- Original, unused condition
- Original packaging with all tags attached
- All accessories and documentation included

### Non-Returnable Items
The following items cannot be returned:
- Gift cards
- Downloadable software
- Personalized/custom items
- Items marked as "Final Sale"
- Opened software or media
- Undergarments and swimwear (for hygiene reasons)

## How to Return

### Online Return Process
1. Log into your account
2. Go to Order History
3. Select the item(s) you wish to return
4. Choose return reason
5. Print prepaid shipping label
6. Package items securely
7. Drop off at any authorized carrier location

### In-Store Returns
If you have a physical store near you:
1. Bring the item with original receipt or order confirmation
2. Visit the customer service desk
3. Returns processed immediately

## Refund Information

### Refund Timeline
- Credit card refunds: 5-10 business days after we receive the return
- PayPal refunds: 3-5 business days
- Store credit: Issued immediately upon return processing

### Refund Amount
- Original item price will be refunded
- Original shipping charges are non-refundable (except for defective items)
- Return shipping is free for defective items only

### Partial Refunds
Partial refunds may be issued for:
- Items not in original condition
- Items returned more than 30 days after delivery
- Items missing components or packaging

## Exchanges

### How to Exchange
1. Return the original item using the process above
2. Place a new order for the desired item
3. Use expedited shipping at no extra cost (code: EXCHANGE)

### Size Exchanges
For clothing and shoe size exchanges:
1. Contact customer support
2. We'll send the new size before receiving your return
3. Use the prepaid label to return the original

## Defective Items

### What Qualifies as Defective
- Manufacturing defects
- Items damaged during shipping
- Items not functioning as described

### Defective Item Process
1. Contact support within 90 days
2. Provide photos or video of the defect
3. We'll provide a prepaid return label
4. Choose replacement or full refund
